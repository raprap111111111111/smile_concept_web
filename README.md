## 🔧 Setup

### 1. Clone & Install
\`\`\`bash
git clone <repo-url>
cd smile_concept_web
flutter pub get
\`\`\`

### 2. Environment Setup
\`\`\`bash
cp .env.example .env
\`\`\`

Then edit `.env` with your backend URL:
- **Laravel Herd:** `http://localhost/api/v1`
- **php artisan serve:** `http://localhost:8000/api/v1`

### 3. Run
\`\`\`bash
flutter run
\`\`\`
